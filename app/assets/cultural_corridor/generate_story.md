# 传统故事生成任务

## 基本信息

- 朝代：{{dynasty}}
- 故事主题：{{theme}}
- 关键词：{{keywords}}

## 请生成一个符合以下 JSON 格式的传统故事：

```json
{
  "title": "故事标题",
  "dynasty": "故事所属朝代",
  "period": "具体时期或年代",
  "theme": "故事主题分类",
  "content": "完整的故事内容，包含开端、发展、高潮和结局，富有教育意义",
  "moral": "故事寓意或教育价值",
  "reference": "故事的来源或参考资料（如适用）"
}
```

## 要求：

- 故事内容应具有历史真实性和文化内涵
- 针对中小学生群体，内容应易于理解
- 故事应包含明确的教育意义或价值观引导
- 叙述生动有趣，能引发学生兴趣
- 故事篇幅适中（800-1200 字）
- 如果指定了朝代，请确保故事背景与该朝代历史文化相符
